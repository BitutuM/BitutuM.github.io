Lab Challenges

Problem Statement: Scrape structured data from a live website and process it into a format suitable for analysis. The goal is to extract data from a website that requires user interaction and clean it for further use.

Approach: Used Python with the libraries Requests and BeautifulSoup to scrape a table from a live website. Cleaned the data using Pandas and stored it into a structured format (CSV).

Tools Used: Python, BeautifulSoup, Requests, Pandas

Key Lessons Learned: Developed a deeper understanding of web scraping techniques, handling dynamic content on websites, and managing large datasets with Pandas. Gained experience in automating data collection processes.

Problem Statement: Clean and transform a Netflix dataset that contains inconsistencies, missing values, and other issues. The goal is to create a clean, structured dataset suitable for analysis and visualization.

Approach: Analyzed the dataset using Pandas to identify missing values, duplicates, and outliers. Performed transformations like date parsing, splitting columns, and handling missing data. Finally, exported the cleaned dataset.

Tools Used: Pandas, Python

Key Lessons Learned: Gained practical experience in data wrangling techniques, handling large datasets, and using data transformation tools. Learned how to structure and clean real-world datasets for further analysis.

Problem Statement: Analyze the Titanic dataset to explore relationships between passenger features and survival rates. The goal is to uncover meaningful patterns and insights through data exploration.

Approach: Conducted univariate, bivariate, and multivariate analysis using Python libraries like Pandas and Matplotlib. Visualized key features such as age, gender, class, and embarkation point to understand their relationship with survival.

Tools Used: Pandas, Matplotlib, Python

Key Lessons Learned: Improved my skills in exploratory data analysis and visualization. Learned how to identify key features that influence outcomes (survival) and how to communicate findings visually.
